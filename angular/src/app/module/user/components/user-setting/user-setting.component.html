User Setting
  • Time Zone - Specify showing all times in UTC time or the local time of your computer.
  • Date Format - Date format for July 28, 2020, see examples below.
    • MM/dd/yyyy 07/28/2020.
    • dd-MMM-yyyy 28-Jul-2020
  • Unsubscribe from Role based Email Notifications - If set to true (blue check), then any email notifications based on any role you are assigned to will not be sent to you. For example, if you're in the Asurion role, and a new LetterCode is created, with this unsubscribe checked, you will not receive an email that the Letter Code was created.
